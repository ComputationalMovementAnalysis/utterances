


# Q & A

![](https://image.freepik.com/free-vector/people-discuss-news-with-each-other-exchange-messages-ideas-speech-bubbles-white-background_110633-625.jpg)


This repository can be used as a place for discussion and questions regarding the exercises hosted on [computationalmovementanalysis.github.io](https://computationalmovementanalysis.github.io).
Each page on said website has a section at the very bottom where you can place your questions through your Github account. Once you place your question there, it will automatically open an issue in *this* repo, but you can also directly [open an issue here](https://github.com/ComputationalMovementAnalysis/utterances/issues).
